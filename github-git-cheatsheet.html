Credentials & Authentication

Personal Access Token (PAT) — use instead of password
  1. Go to GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Click Generate new token → check repo scope → copy the token
  3. When terminal prompts for password, paste the token
Cache credentials so you don't re-enter them
# Cache for 1 hour
git config --global credential.helper cache

# Store permanently via macOS Keychain
git config --global credential.helper osxkeychain

GitHub Actions — Self-Hosted Runner

Register your machine as a runner
  1. Go to your repo → Settings → Actions → Runners → New self-hosted runner
  2. Select macOS and follow the install commands in the UI
  3. Run ./run.sh on your machine to start listening
  4. The workflow triggers automatically on every push to main
